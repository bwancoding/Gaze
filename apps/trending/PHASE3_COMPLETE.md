**核心算法**:
```python

2. TF-IDF 向量化 → 计算词项权重
3. 余弦相似度 → similarity = (A·B) / (||A|| × ||B||)
4. 聚类决策 → 相似度>=0.6 归入现有事件，否则创建新事件
5. 增量更新 → 实时聚类 + 定期合并重复
```

## 📝 技术总结

### 热度算法设计

**时间衰减**:
- 选择指数衰减：`e^(-λ×hours)`
- λ=0.1（每小时衰减约 10%）
- 符合新闻传播规律

**互动权重**:
- 使用对数缩放：`log10(x+1)`
- 避免极端值主导
- 评论权重 5，分享权重 3

**源优先级**:
- P0=1.5（权威媒体）
- P1=1.2（主流媒体）
- P2=1.0（其他媒体）

### 事件聚类设计

**文本表示**:
- TF-IDF 向量化
- 标题权重×2
- 移除停用词

**相似度计算**:
- 余弦相似度
- 阈值 0.6（可配置）
- 支持增量更新

**聚类策略**:
- 新文章检测→现有事件 or 新事件
- 定期检测重复事件
- 自动合并相似事件
